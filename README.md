# 🎙️ InterviewMate — AI-Powered Mock Interview Platform

**InterviewMate** is a web application that helps job seekers practice for real-world interviews using dynamic AI-generated questions, instant feedback, scoring, and improved answers — all without any setup hassle or payment.

> 💡 Practice smarter. Get better. Feel confident.

---

## 🚀 Live Demo (if available)

> https://interview-mate-e61ad.web.app

---

## 🔍 Features

- 🤖 **AI-generated interview questions** (no hardcoded Q&A)
- 📊 **Real-time scoring (1–10)** with actionable feedback
- ✍️ **Improved answer generation** (STAR-based where applicable)
- 🔐 **Secure authentication** via Firebase (Email/Password + Google)
- 📈 **Mock interview history & progress tracking**
- ⚡ **Lightweight, fast, and mobile-responsive UI**

---

## 🧠 How the AI Works

1. **Dynamic Question Generation**  
   The AI creates relevant interview questions on the fly based on user context.

2. **Answer Evaluation**  
   Each user answer is analyzed and given:
   - A score from 1 to 10
   - Clear, constructive feedback
   - An improved, well-structured answer

3. **Real Interview Simulation**  
   No static Q&A — just a realistic, adaptive interview experience.

## 🛠️ Tech Stack

| Area               | Technology                                 |
|--------------------|--------------------------------------------|
| Frontend           | HTML, CSS, Vanilla JavaScript              |
| Authentication     | Firebase Auth (Email/Google)               |
| Backend / API      | Cloudflare Workers (serverless)            |
| AI Provider        | Groq API (LLaMA models, fast inference)    |
| Data & History     | Designed for Firestore (ready for integration) |

> 🔒 API keys are secured via Cloudflare Workers — never exposed to the frontend.
